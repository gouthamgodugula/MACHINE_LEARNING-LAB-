# MACHINE_LEARNING-LAB-
This repository contains the implementation of various **Machine Learning algorithms** performed as part of the **Machine Learning Laboratory course**. The programs are implemented using **Python** and executed using **Google Colab / Jupyter Notebook**.
## 📌 Objectives
* To understand the working of basic Machine Learning algorithms.
* To implement supervised learning algorithms using Python.
* To analyze the performance of different machine learning models.
## 🛠 Tools and Technologies

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook / Google Colab
